# Heroization

MVP-сайт для стримеров: собираем список усложнений, крутим рулетку по весам и получаем готовый `overlay` для OBS без бэкенда и без авторизации.

## Что уже есть

- редактор усложнений с текстом и суммой/весом;
- рулетка в формате pie-chart с анимированным вращением;
- пул выпавших усложнений;
- генерация отдельной страницы `/overlay`, которую можно использовать как Browser Source;
- вся логика живет только на клиенте, без сервера и без сохранения.

## Локальный запуск

```bash
npm install
npm run dev
```

## Деплой на Vercel

1. Импортировать репозиторий в Vercel.
2. Framework Preset: `Next.js`.
3. Build Command оставить по умолчанию: `next build`.
4. После деплоя использовать ссылку вида `https://your-domain.vercel.app/overlay?...` как Browser Source в OBS.

## Как работает overlay

Выбранные усложнения сериализуются в URL-параметр `data`, поэтому на текущем этапе не нужен ни бэкенд, ни база. Это удобно для MVP, но позже можно заменить на `localStorage`, серверное хранение или share-ссылки с коротким ID.
